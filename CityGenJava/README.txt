LAUNCH COMMANDS:
To launch a server:
	Enter a string config file (e.g. realfloorconfig.txt) with no additional parameters to
	launch a game server.
To launch a client:
	Enter a string config file (e.g. realfloorconfig.txt) with an IP parameter,
	or "LOCAL" to connect to the local host.

Key Controls:

WASD - movement
Q : toggle equip left item
E : toggle equip right item

U : Use equipped item
T : Drop equipped item

Mouse controls:
Move mouse left or right to look in the moved direction
Click to pick up nearby item

General game:
Click on ladders to go up a floor, portals to go down a floor, and
keycards to open doors
